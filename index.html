Credit card fraud detection with XGBoost and SMOTE

Developed a machine learning model using XGBoost to accurately detect fraudulent credit card transactions. To address class imbalance, applied SMOTE (Synthetic Minority Over-sampling Technique), significantly improving the model’s recall and F1-score for minority fraud cases. Emphasized model interpretability and performance evaluation using precision-recall metrics and ROC curves.

Customer churn prediction using AutoML

Built a machine learning model to predict loan defaults using XGBoost, focusing on accuracy and reliability. Applied Explainable AI techniques like SHAP and LIME to interpret model predictions, helping uncover key factors behind default risk. Emphasized transparency and trust in financial decision-making through feature importance analysis and clear visual explanations
